# Sample skill

This is a **project skill** at `.cursor/skills/sample-skill/`. Treat it as a copy-paste starting point: duplicate the folder, rename it, and replace the body with real instructions.

## What to edit

1. **Folder name** — Use lowercase words separated by hyphens (64 chars max).
2. **Frontmatter `name`** — Must match the folder name.
3. **Frontmatter `description`** — Third person; say what the skill does and when to use it (include trigger terms).
4. **Body** — Short steps the agent should follow; link to `reference.md` or `examples.md` only if you need more detail.

## Workflow (replace with your own)

1. Confirm the user’s goal.
2. Perform the minimal change that satisfies it.
3. Run the project’s tests if the change touches behavior.

## Optional files

Add alongside `SKILL.md` when needed: `reference.md`, `examples.md`, or `scripts/` for utilities. Keep `SKILL.md` under about 500 lines; put long material in those files.
